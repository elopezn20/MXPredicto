## 이것은 무엇인가요?

FIFA 월드컵 2026 예측 게임입니다. 각 참가자는 마감 전에 104경기의 정확한 스코어를 예측합니다. 대회가 끝날 때 가장 많은 점수를 획득한 참가자가 우승합니다.

---

## 참여 방법

이 게임은 **초대 전용**입니다. 관리자가 등록 링크가 포함된 이메일을 보내드립니다. 해당 링크에서 이름과 비밀번호를 설정하면 등록이 완료됩니다. 등록 후에는 언제든지 위의 링크에서 로그인할 수 있습니다.

각 참가자는 최대 한 개의 계정만 가질 수 있습니다 — 중복 가입은 허용되지 않습니다.

## 상금

총 상금은 **$2,500,000 CLP**이며, 최종 순위 상위 8명에게 분배됩니다.

| 순위 | 상금 (CLP) |
|---|---|
| 1위 | $1,250,000 |
| 2위 | $500,000 |
| 3위 | $250,000 |
| 4위 | $150,000 |
| 5위 | $125,000 |
| 6위 | $100,000 |
| 7위 | $75,000 |
| 8위 | $50,000 |
| **합계** | **$2,500,000** |

### 동점 시 상금 배분

두 명 이상이 점수, 적중 수, 영점 수가 모두 같아 동점인 경우, 해당 참가자들이 차지한 순위의 상금 합계를 균등하게 나누어 받습니다.

**예시:**

- 1위 2인 동점 → 각자 **($1,250,000 + $500,000) / 2 = $875,000**.
- 1위 3인 동점 → 각자 **($1,250,000 + $500,000 + $250,000) / 3 ≈ $666,667**.
- 8위 2인 동점 (마지막 상금 순위) → 각자 **($50,000 + $0) / 2 = $25,000**.

## 예측 방법

### 경기 예측

- 각 경기의 **정확한 스코어**를 예측합니다 (예: 아르헨티나 2 – 브라질 1).
- 토너먼트 경기에서 정규시간/연장전 후 동점일 경우 **진출 팀**(승부차기 승자)도 선택해야 합니다.
- 마감 전까지 **원하는 만큼** 저장하고 수정할 수 있습니다.

### 라운드 마감

> **라운드는 해당 라운드 첫 경기 당일 칠레 시간 00:00에 마감됩니다.**

한 라운드의 첫 경기가 6월 15일 월요일(칠레 시간)에 있다면, 해당 라운드의 모든 예측은 칠레 시간 6월 15일 월요일 00:00(자정)에 잠깁니다 — 실제 경기는 그날 더 늦은 시각에 진행되더라도 마찬가지입니다.

마감 후에는 예측을 수정하거나 추가할 수 없습니다. 예측이 없는 경기는 0점입니다. 앱은 다음 마감까지의 카운트다운을 표시합니다.

### 보너스 포디엄

- 월드컵의 **챔피언** (50점), **준우승** (25점), **3위** (15점)를 예측합니다. 최대 합계: **90점**. 각 순위는 독립적으로 채점됩니다 — 챔피언만 맞혀도 다른 두 개가 틀려도 50점을 얻습니다.
- 세 팀은 모두 달라야 합니다.
- 제출 기간은 **조별리그 기간** (대회 첫날부터 가능)입니다. 32강 첫 경기가 시작되면 보너스 포디엄이 영구적으로 마감됩니다.
- 마감 전까지 **원하는 만큼** 저장하고 수정할 수 있습니다.

## 경기당 점수

### 조별리그 (경기당 최대 10점)

| 항목 | 점수 |
|---|---|
| 정확한 결과 (승자 또는 무승부) | 5 |
| 홈 팀 정확한 득점 | 2 |
| 원정 팀 정확한 득점 | 2 |
| 정확한 골득실차 (예측한 방향으로) | 1 |

점수는 누적됩니다. 완벽한 예측 = **10점**. 홈과 원정의 정확한 득점은 전체 결과가 틀려도 점수를 얻습니다.

> **골득실차 관련:** 골득실차는 크기와 *방향* 모두 일치해야 합니다. 2–1 (홈 1점 차 승)을 예측했다면, 결과가 1–2나 2–3이면 득실차도 1이지만 이 점수를 얻지 못합니다 — 승자가 다르기 때문입니다.

**예시:**

| 예측 | 실제 결과 | 점수 | 이유 |
|---|---|---|---|
| 2–1 | 2–1 | **10** | 완벽 — 결과 ✓ (5), 홈 ✓ (2), 원정 ✓ (2), 득실+승자 ✓ (1) |
| 2–0 | 3–1 | **6** | 결과 ✓ (5) + 득실+승자 ✓ (1); 득점 불일치 |
| 1–1 | 2–2 | **6** | 무승부 ✓ (5) + 득실 0 ✓ (1); 득점 불일치 |
| 3–1 | 2–1 | **7** | 결과 ✓ (5) + 원정 정확 ✓ (2); 홈 틀림, 득실 다름 |
| 1–0 | 1–0 | **10** | 완벽 |
| 2–1 | 2–0 | **7** | 결과 ✓ (5), 홈 ✓ (2), 원정 ✗; 득실 다름 (1 ≠ 2) |
| 0–1 | 0–2 | **7** | 결과 ✓ (5), 홈 ✓ (2), 원정 ✗; 득실 다름 |
| 1–1 | 1–0 | **2** | 결과 ✗ (무승부 예측, 홈 승); 홈 ✓ (2); 원정 ✗ |
| 2–1 | 1–2 | **0** | 결과 틀림 — 점수 없음 |
| 0–0 | 0–0 | **10** | 완벽 |

> **무승부 관련:** 무승부를 예측하고 경기가 무승부로 끝나면, 득점이 달라도 골득실 보너스 점수를 얻습니다 (둘 다 득실차 0이고 "승자" = 무승부).

### 토너먼트 (경기당 최대 25점)

| 항목 | 점수 |
|---|---|
| 정규시간 또는 연장전 후 정확한 결과 | 10 |
| 홈 팀 정확한 득점 | 4 |
| 원정 팀 정확한 득점 | 4 |
| 정확한 골득실차 (예측한 방향으로) | 2 |
| 진출 팀 | 5 |

완벽한 예측 = **25점**.

> **스코어 관련:** 연장전 골은 최종 스코어에 포함됩니다. 90분에 1–1이고 홈 팀이 연장전에 골을 넣으면 공식 결과는 **2–1**입니다. 연장전 후에도 동점이면 승부차기가 진행됩니다.

> **결과와 무관한 득점:** 홈 또는 원정 득점을 정확히 맞히면 전체 결과가 틀려도 점수를 얻습니다.

> **골득실차 관련:** 골득실차는 크기와 *방향* 모두 일치해야 합니다. 2–1 (홈 1점 차 승)을 예측했다면, 결과가 1–2나 2–3이면 득실차도 1이지만 이 기준 점수를 얻지 못합니다.

> **진출 팀 — 방법 무관:** 이 항목은 어느 팀이 다음 라운드에 진출했는지만 봅니다. 정규시간, 연장전, 승부차기 여부는 무관합니다. 예측한 팀이 실제 진출 팀과 일치하면 정확한 스코어와 무관하게 5점을 얻습니다.

**예시:**

| 예측 | 실제 결과 | 진출 | 점수 | 내역 |
|---|---|---|---|---|

| 원정 0–2 | 원정 0–2 | 원정 | **25** | 결과✓10, 홈✓4, 원정✓4, 득실✓2, 진출✓5 |
| 1–1 + 홈 | 1–1 | 홈 | **25** | 결과✓10, 홈✓4, 원정✓4, 득실✓2, 진출✓5 |
| 1–1 + 홈 | 1–1 | 원정 | **20** | 결과✓10, 홈✓4, 원정✓4, 득실✓2, 진출✗0 |
| 1–1 + (미선택) | 1–1 | 홈 | **20** | 결과✓10, 홈✓4, 원정✓4, 득실✓2, 진출✗0 — 승부차기 팀 선택 필요 |
| 홈 2–0 | 홈 2–1 | 홈 | **19** | 결과✓10, 홈✓4, 원정✗0, 득실✗0, 진출✓5 |
| 홈 3–0 | 홈 1–0 | 홈 | **19** | 결과✓10, 홈✗0, 원정✓4, 득실✗0, 진출✓5 |
| 홈 1–0 | 홈 2–1 | 홈 | **17** | 결과✓10, 홈✗0, 원정✗0, 득실✓2, 진출✓5 |
| 홈 2–1 | 1–1 + 원정 | 원정 | **4** | 결과✗0, 홈✗0, 원정✓4, 득실✗0, 진출✗0 |
| 홈 2–1 | 1–1 + 홈 | 홈 | **9** | 결과✗0, 홈✗0, 원정✓4, 득실✗0, 진출✓5 |

### 보너스 포디엄 (최대 90점)

| 항목 | 점수 |
|---|---|
| 챔피언 적중 | 50 |
| 준우승 적중 | 25 |
| 3위 적중 | 15 |

각 순위는 독립적으로 채점됩니다. 나머지가 틀려도 챔피언만 맞히면 50점을 얻을 수 있습니다.

## 순위표

순위는 **총 점수** (경기 + 포디엄) 기준으로 정렬됩니다.

**동점 처리 (순서):**
1. "적중" — 해당 경기 최대 점수를 획득한 경기 수가 많은 순. 조별리그 **10점**, 토너먼트 **25점**.
2. 0점 경기 수("영점") 적은 순 — 예측이 없는 경기는 0점으로 계산됩니다. 부분 점수 (0점 초과이지만 최대 점수 미만)는 적중도 영점도 아닙니다.

순위표에는 순위, 이름, 점수, 적중 수, 영점 수, 선두와의 점수 차가 표시됩니다.

---

## 예측 공개 여부

- 예측은 **라운드가 마감될 때까지 비공개**입니다.
- 라운드가 마감되면 해당 경기들의 모든 참가자 예측이 공개됩니다.
- **보너스 포디엄** 역시 포디엄 마감 시점 (32강 첫 경기가 시작되는 시각)까지 비공개입니다. 마감 후에는 모든 참가자의 포디엄 예측이 공개됩니다.

## 자주 묻는 질문

**예측을 변경할 수 있나요?**
네, 라운드 마감 전까지 원하는 만큼 변경할 수 있습니다. 마지막으로 저장된 버전이 유효합니다.

**경기를 예측하지 않으면 어떻게 되나요?**
해당 경기는 0점 처리됩니다.

**보너스 포디엄은 어떻게 되나요?**
마감 전까지 원하는 만큼 저장하고 수정할 수 있습니다. 마지막으로 저장된 버전이 유효합니다.

**각 라운드 마감 시각은 어떻게 알 수 있나요?**
앱이 항상 카운트다운을 표시합니다. 각 라운드 마감은 해당 라운드 첫 경기 당일 칠레 시간 00:00입니다 — 실제 킥오프 시각이 아니라 라운드가 시작되는 날의 자정입니다.
